This page showcases async tags interacting in the three/four different ways that display() can be used:

import asyncio for i in range(1,6): display(f"A{i}", targetID = "explicit-target-div") await asyncio.sleep(.1) import asyncio for i in range(1,6): display(f"B{i}", targetID = "explicit-target-div") await asyncio.sleep(.1)
#Explicit Target Div (A & B)
import asyncio for i in range(1,6): display(f"C{i}") await asyncio.sleep(.1) import asyncio for i in range(1,6): display(f"D{i}") await asyncio.sleep(.1)
#Output-Target Div (C & D)
